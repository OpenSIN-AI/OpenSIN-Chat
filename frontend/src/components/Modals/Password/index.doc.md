# Password modal

## Purpose

Provides the authentication shell shown before a user enters OpenSIN Chat. The layout keeps product context in a compact side panel on desktop while prioritizing the sign-in action.

## Design notes

- Uses the shared neutral dark palette and avoids decorative gradients.
- Secondary copy meets the same visible contrast tier as other workspace chrome.
- The context panel is intentionally hidden below the `lg` breakpoint so sign-in remains focused on smaller screens.
